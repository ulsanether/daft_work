This Megapack is a collection of sample resources which can help you when finding your way round Gensole for the first time.

Examples :

1. Bundles* (For Tab 1).
2. Shoe Profiles (For Tab 2).
3. Foot Scans (For Tab 3).
4. DXF Density Zones (For Tab 4 - optional).
5. Slic3r AMFS, conmfigurations and example gcode files (For Slic3r).

*A Gensole Bundle is a “save” file created in Tab 4 of Gensole and can be loaded into Tab 1
